Πρωτού προχωρήσετε στην εγκατάσταση του Oracle VM Virtual Box:
Θα πρέπει να γνωρίζετε το μοντέλο του επεξεργαστή σας ώστε να ελέγξετε αν υποστηρίζει την προσομοίωση λειτουργικών.
Αν έχετε Windows 7 ή 8 :

  • Από τις ιδιότητες του υπολογιστή βλέπετε το μοντέλο του επεξεργαστή.
Αν έχετε Windows 10 :
  • Από τις ιδιότητες του υπολογιστή βλέπετε το μοντέλο του επεξεργαστή.

    • - Ένας άλλος τρόπος :

    • Από την αναζήτηση αν γράψετε πληροφορίες συστήματος μπορείτε να δείτε το μοντέλο του επεξεργαστή.

Στην συνέχεια, θα πρέπει να αναζητήσετε αν το μοντέλο του επεξεργαστή σας υποστηρίζει την χρήση εικονικού λειτουργικού συστήματος.Αν ναι, τότε θα πρέπει να σιγουρευτείτε ότι αυτή η λειτουργία είναι ενεργοποιημένη από τις ρυθμίσεις BIOS του υπολογιστή σας.


Για να εγκαταστήσετε το Oracle VM Virtual Box (επιλέξτε την τελευταία έκδοση) θα χρειαστείτε να ελέγξετε ποια έκδοση λειτουργικού συστήματος έχει ο υπολογιστής. (x64 ή x86)


Αν έχετε Windows 7 ή 8 :
  • Από τις ιδιότητες του υπολογιστή βλέπετε την έκδοση του λειτουργικού συστήματος.
Αν έχετε Windows 10 :
  • Από τις ιδιότητες του υπολογιστή βλέπετε την έκδοση του λειτουργικού συστήματος.

    • -Ένας άλλος τρόπος:

    • Από την αναζήτηση αν γράψετε πληροφορίες συστήματος μπορείτε να δείτε την έκδοση του λειτουργικού συστήματος.


Θα χρειαστείτε να έχετε κατεβάσει κάποιο λειτουργικό σύστημα (βασισμένο σε UNIX π.χ. Ubuntu με LTS: Long Term Support - Μακροχρόνια Υποστήριξη) ώστε να προχωρήσετε στην εγκατάσταση του στο πρόγραμμα VmWare.


Αφού έχετε βεβαιωθεί ότι πληρείτε όλες τις ανωτέρω προυποθέσεις τότε μπορείτε να υποστηρίξετε την χρήση ενός λειτουργικού μέσω της μηχανής Oracle VM Virtual Box. Αν δεν υποστηρίζεται κάποια προυπουπόθεση τότε υπάρχουν λύσεις!
Στην συνέχεια μπορείτε να προχωρήσετε με την εγκατάσταση του προγράμματος όπου θα τρέχει τα Ubuntu.
